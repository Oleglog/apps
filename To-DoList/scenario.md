## 1 Добаление таска:
# Пользоваетль входит интерфес приложения -> выбирает "создание таска" -> вводит заголовок и описание задачи -> сохраняет задачу

## 2 Изменение таска
# Пользваотель входит интерфес приложения -> выбирает "изменение таска" -> вводит новые значения для заголовка и описания -> применят изменеия.

## 3 Удаление таска
# Пользваотель входит интерфес приложения -> выбирает "удаление таска" -> Выбирает нужный таск дял удаления -> удаляет его.

## 4 Изменение статуса таска на "Выполнен"

## 5 Изменение статуса таска на "Не выполнен"
# Пользваотель входит интерфес приложения -> выбирает "Все таски" -> Выбирает нужный таск для изменения статуса -> Помечает как "Не выполнен" -> Применяет изменения.
